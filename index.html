Explanation

説明

This tool helps configure *GITADORA GALAXY WAVE DELTA Drummania* settings to fix the SHUTTER (OUT) top as the judgment line, minimizing visual note jitter. Due to the no-JS constraint, calculations must be done manually. The green number (pseudo-BPM adjustment, similar to IIDX) maintains consistent note spacing. For BPM=180, HI-SPEED=x3.75, Note Display Adjust=-6 is optimal. For other BPMs, use spline interpolation between limits (BPM=78: HI-SPEED=x8.5; BPM=400: HI-SPEED=x1.5). Formula: Green Number = (60000 / BPM) * HI-SPEED * (Refresh Rate / 60). Adjust HI-SPEED, Note Display Adjust, Judgment Line Position, and SHUTTER (IN) to maintain this value when BPM changes. Monitor delay (9ms) is compensated by Note Display Adjust.

このツールは、*GITADORA GALAXY WAVE DELTA ドラムマニア*の設定を調整し、シャッター（アウト）の上部を判定ラインとして固定し、ノーツのブレを最小限に抑えるのに役立ちます。JSを使用できないため、計算は手動で行う必要があります。緑数字（IIDXに似た擬似BPM調整）は、ノーツの間隔を一定に保ちます。BPM=180の場合、ハイスピード=x3.75、ノーツ表示調整=-6が最適です。他のBPMでは、限界値（BPM=78：ハイスピード=x8.5、BPM=400：ハイスピード=x1.5）の間でスプライン補間を使用します。計算式：緑数字 = (60000 / BPM) * ハイスピード * (リフレッシュレート / 60)。BPM変更時にこの値を維持するよう、ハイスピード、ノーツ表示調整、判定ライン位置、シャッター（イン）を調整します。モニター遅延（9ms）はノーツ表示調整で補正されます。

Notes

注意書き

- Konaste HI-SPEED adjusts in 0.5 units.
- Assumes initial monitor delay of 9ms.
- Sound depth varies by song; use headphones for zero audio delay.
- HI-SPEED: x3.75 at BPM: 180 is the limit for stable notes.

- コナステのハイスピード調整は0.5単位です。
- 初期出荷モニターの遅延は9msと仮定。
- 曲によってスピーカー音の奥行バランスが異なるため、ヘッドホン装着で音声遅延なしを想定。
- ハイスピード：x3.75、BPM：180はノーツがブレない限界です。
